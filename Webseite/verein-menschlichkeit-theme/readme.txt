== Beschreibung ==
Modernes, barrierefreies und CiviCRM-optimiertes WordPress-Theme für Vereine.

== Features ==
* Responsive Design
* Barrierefreiheit (Skiplinks, ARIA, Fokus)
* CiviCRM-Integration (eigene Styles, Templates, JS)
* Custom Logo, Beitragsbilder, Übersetzbarkeit
* Hauptmenü, Suchfeld, Mobile-Navigation

== Installation ==
1. Theme als ZIP hochladen
2. Aktivieren
3. Menü unter Design > Menüs zuweisen
4. Logo im Customizer setzen (optional)
